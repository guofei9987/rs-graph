# rs-graphs
Graph algorithms in Rust

Rust 实现图数据库

功能
- 增
    - 增加节点 add_node
    - 增加边 add_edge
- 查
    - get_node_by_idx
    - get_name_by_idx
    - [ ] 根据各种条件做查询
- 删
    - 删除节点，del_node_by_idx，del_node_by_name，
    - 删除边，del_edge_by_idx，del_edge_by_name
    - clear:清空所有数据
- 改
    - 改节点对应的值：add_node
- 高级功能
    - 遍历全部下游
    - 寻找最近链路
    - [ ] 寻找关键链接
